# Data Sources

## Raw Data

### anomaly_locations.csv

- Source: OpenAQ API
- Download date: 2024-XX-XX
- Description: Station locations for anomaly detection challenge
- Format: CSV, columns: location_id, latitude, longitude, country

### anomaly_measurements.csv

- Source: OpenAQ API
- Download date: 2024-XX-XX
- Description: PM2.5 hourly measurements
- Format: CSV, columns: location_id, date, value, pollutant, unit

## How to download fresh data

The dataset is not publicly available for direct download.

To obtain access, please submit the application form provided by CREA:

https://energyandcleanair.org/challenge/anomaly-detection/

After approval, you will receive instructions and access to the challenge dataset.
